Инструкция по добавлению изображения для бота:

1. Подготовьте изображение с логотипом или иллюстрацией для Enter Tier 2
2. Назовите файл "tier2_logo.jpg" (или измените имя файла в коде bot.py)
3. Поместите файл в директорию "images/"
4. Перезапустите бота

Рекомендации по изображению:
- Формат: JPG или PNG
- Размер: не более 1280x1280 пикселей
- Вес: не более 5 МБ (ограничение Telegram)
- Соотношение сторон: лучше всего 1:1 (квадратное) или 16:9

После добавления изображения, бот будет отправлять его вместе с приветственным сообщением при команде /start.
